Create a React application that consists of the following components:

App (main component)
ProductList (displays a list of products)
Product (represents a single product)
Basket (displays the items added to the shopping cart)



Functionality:
==============

Add items to the shopping cart.
Remove items from the shopping cart.
View the current items in the shopping cart.
Clear the shopping cart.

Structure of data
=================
const products = [
  { id: 1, name: 'Product 1', price: 10 },
  { id: 2, name: 'Product 2', price: 20 },
  { id: 3, name: 'Product 2', price: 20 },
];
